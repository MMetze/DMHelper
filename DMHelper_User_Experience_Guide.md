# DMHelper User Experience Guide

## Overview

DMHelper is a comprehensive desktop application designed for tabletop RPG Dungeon Masters (DMs) and Game Masters (GMs) to manage campaigns, characters, battles, maps, and all aspects of tabletop role-playing game sessions. The application supports multiple RPG systems through its template-based architecture.

---

## 1. Campaign Management

### Creating and Managing Campaigns
- **New Campaign**: Create a new campaign by selecting your preferred ruleset (D&D 5e, Pathfinder, etc.)
- **Save/Load Campaigns**: Save your work and load existing campaigns with full state preservation
- **Campaign Tree**: Organize all campaign content in a hierarchical tree structure for easy navigation
- **Campaign Notes**: Add and manage campaign-wide notes and documentation
- **Archive/Unarchive**: Organize older content by archiving completed adventures or unused materials

### Import/Export Functionality  
- **Campaign Export**: Export entire campaigns for sharing or backup
- **Selective Import**: Import specific elements from other campaigns
- **Backup Management**: Create and restore campaign backups

---

## 2. Character and Party Management

### Character Creation and Management
- **Template-Based Character Sheets**: Dynamic character sheets that adapt to any RPG system (D&D 5e, Pathfinder, etc.)
- **Drag-and-Drop Character Creation**: Import character data from various sources
- **D&D Beyond Integration**: Direct import of characters from D&D Beyond with automatic syncing
- **HeroForge Integration**: Import character data and assets from HeroForge
- **Custom Character Fields**: Add system-specific attributes and fields as needed

### Party Management
- **Party Composition**: Create and manage adventuring parties
- **Character Status Tracking**: Monitor health, resources, and status effects for all party members
- **Party Overview**: Quick reference view of all party member statistics and current status
- **Active/Inactive Status**: Toggle characters between active party members and inactive NPCs

### Character Features
- **Hit Points and Health Tracking**: Full HP management including temporary HP
- **Spell Slot Management**: Visual spell slot tracking with automatic restoration
- **Equipment and Inventory**: Comprehensive gear and item management
- **Character Portraits**: Add and manage character artwork and tokens
- **Custom Counters**: Create trackable resources (sorcery points, ki points, etc.)

---

## 3. Battle and Combat System

### Battle Encounter Management
- **Battle Creation**: Set up combat encounters with maps, participants, and environmental factors
- **Initiative Tracking**: Automated initiative order with optional "acted" flags for each combatant
- **Generalized Initiative**: Support for both individual and group initiative systems
- **Turn Management**: Navigate through combat rounds with clear turn indicators

### Combat Participants
- **Player Characters**: Add party members to combat with full stat integration
- **Monsters and NPCs**: Place enemies from the bestiary or create custom combatants
- **Token Management**: Visual tokens on battle maps with drag-and-drop positioning
- **Health Tracking**: Real-time HP tracking for all combatants
- **Status Effects**: Apply and track conditions, buffs, debuffs, and other effects
- **Combatant Widgets**: Individual character/monster displays with quick reference stats

### Battle Map Features
- **Grid Systems**: Square or hex grid support with snap-to-grid token movement
- **Distance Measurement**: Measure distances between tokens and locations
- **Line of Sight**: Visual line-of-sight calculations
- **Area Effects**: Create visual spell areas (cones, spheres, lines, cubes)
- **Token Visibility**: Show/hide tokens from players during combat
- **Battle History**: Combat log tracking damage, actions, and events

---

## 4. Mapping and Visual Tools

### Map Management
- **Map Creation**: Create world maps and battle maps from image files
- **Multiple Map Types**: Support for world maps, battle maps, and reference maps
- **Map Layers**: Organize map elements using multiple layers (background, tokens, effects, etc.)
- **Map Markers**: Add interactive markers for locations, NPCs, and points of interest

### Fog of War System  
- **Dynamic FOW**: Real-time fog of war management during sessions
- **FOW Tools**: Paint, erase, and shape fog of war areas
- **Player Vision**: Control what areas players can see
- **Token-Based Vision**: Automatic FOW removal based on character movement (light source simulation)

### Visual Effects and Layers
- **Layer System**: Organize visual elements in separate, manageable layers
- **Effect Overlays**: Add visual effects for spells, environmental hazards, etc.
- **Video Integration**: Embed video content as animated map elements
- **Custom Graphics**: Import and use custom artwork and assets

---

## 5. Text and Story Content

### Text Encounters
- **Rich Text Editor**: Create formatted story content with full text styling
- **Hyperlink Support**: Link between different campaign elements
- **Text Animation**: Scrolling text with customizable speed for dramatic reveals
- **Background Images**: Add atmospheric images to text encounters
- **Text Publishing**: Display formatted text to players via the publish window

### Linked Text Content
- **File Linking**: Link external HTML, Markdown, or plain text files
- **Dynamic Content**: Automatically update when linked files change
- **Format Support**: Native support for HTML, Markdown, and plain text formats
- **Code View**: Toggle between formatted and raw text views

### Formatting Features
- **Font Controls**: Bold, italic, underline, and color options
- **Alignment**: Left, right, center, and justified text alignment
- **Lists and Structure**: Bullets, numbering, and hierarchical content
- **Custom Styling**: Advanced formatting options for professional presentation

## Getting Started

1. **Create Your First Campaign**: Start with the New Campaign dialog and select your preferred RPG system
2. **Import Characters**: Use the D&D Beyond integration or create characters manually
3. **Add Content**: Create maps, encounters, and story content using the campaign tree
4. **Set Up Your First Battle**: Create a battle encounter with a map and add participants
5. **Explore Publishing**: Use the publish window to share content with your players
6. **Customize Your Workflow**: Adjust settings and shortcuts to match your DM style

DMHelper is designed to be intuitive for new users while providing powerful advanced features for experienced DMs. The modular design allows you to use only the features you need, whether you're running a simple story-focused session or a complex tactical combat encounter.
